## Commands:
1. Run to install dependencies:
```bash
poetry install 
```
2. Runs the program:
``` bash
poetry run invoke start
```
---

Testing:
```bash
poetry run invoke test
```

Coverage report:
```bash
poetry run invoke coverage-report
```
---

Lint:
```bash
poetry run invoke lint
```
